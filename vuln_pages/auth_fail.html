What?

Authentication Failures

Occurs when a system does not properly verify a user's identity, allowing an attacker to gain access as a legitimate user.


Why?

Authentication weaknesses are a common entry point for attackers. Once initial access is gained, it often leads to priviledge escalation. Moving from basic user or guest access to administrative control. If a flaw exists on a lower level of access, it is likely present elsewhere within the system, allowing attackers to maintain long-term access.


How?

This vulnerability occurs when authentication mechanisms are improperly implemented or enforced. Such as weak passwords, lack of validation, or missing security checks. In order to prevent this, systems should enforce strong authentication practices, including secure password handling and complex password enforcement, and proper user verification, such as Multi-Factor Authentication.
