About

WAVL (Web Application Vulnerability Lab) is an interactive educational environment designed to demonstrate common web application vulnerabilities based on the OWASP Top Ten (2025). The platform allows users to explore how these vulnerabilities work and understand how they can be prevented in a real-world applicaiton


The lab is distributed as a preconfigured virtual machine to ensure a consistent and safe environment. This allows user to interact with an intentionally vulnerable systems without risking their own personal machine or larger network. Using localhost in order to keep the system isolated from the outside. The virtual machine also allows removes the need for a complex installation and configuration.


WAVL is designed for students and beginners in cybersecurity, as well as anyone interested in learning how web vulnerabilities function and how to secure against them.

Installation

  • Ensure a form of hypervisor is installed on your machine, free options include Oracle VirtualBox, VMware Workstation Pro, or Hyper-V (Windows only).

    Only one hypervisor should be active at a time

  • Download the virtual machine from the Google Drive

    Files within the virtual machine can be viewed in the Github Repository

  • Import the virtual machine into the hypervisor and run it

  • After the machine boots, open your local browser and go to http://localhost/
